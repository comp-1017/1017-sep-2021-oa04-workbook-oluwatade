Create navigational links within a website

There are several techniques to navigating within a website. The semantic tags for navigating is the <Nav> tag. You can navigate through your website by using relative addressing. For our examples we will use the unorder list tag (<ul>). Our example will consist of:

  • page to page within the same folder
  • Page to page within folders
  • Staying on the current page
  • Going to a specific place on the current page(on page navigation)
  • Going to a specific place on another page (Choosing Apples)

Navigational menus are placed in your <header> semantic area. They may also be placed in your <footer> area.
